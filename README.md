# VoxyProj - Dark Web Puzzle Game

A mysterious dark/deep web themed puzzle game where players must solve challenges to progress through different levels.

## 🎮 Game Overview

This is an interactive web-based puzzle game with a hacker/deep web aesthetic. Players must solve puzzles to gain access to different levels of the system.

## 🕹️ How to Play

1. Open `index.html` in your web browser
2. Read the mission briefing carefully
3. Solve the puzzle to find the access code
4. Enter the code in the access panel
5. Progress to the next level!

## 📋 Mission 01 Solution

**Puzzle:** The hint shows binary code that needs to be decoded.

**Binary Message:** `01010111 01001000 01000001 01010100 00100000 01001001 01010011 00100000 00110010 00101011 00110010 00111111`

**Translation:** "WHAT IS 2+2?"

**Answer:** `4`

Enter **4** in the access code panel to proceed to Level 2!

## 🎨 Features

- **Dark/Deep Web Theme:** Authentic hacker terminal aesthetic with green text on black background
- **Visual Effects:**
  - Scanline CRT monitor effect
  - Screen noise/static
  - Glitch text animations
  - Flickering terminal effect
  - Pulsing and blinking elements
- **Interactive Elements:**
  - Code input validation
  - Error messages with shake animations
  - Success animations
  - Auto-redirect on correct code
  - Random IP address generation
- **Responsive Design:** Works on desktop and mobile devices
